# 專案指引

## 摘要

閩南話傳自遠古。隨著歷史長河推波向前，朝代興衰更迭，此語不復作為朝堂雅言，退化成一地方音，不僅日漸式微，距離消亡似乎僅只是一步之遙；關心此語傳承人士，不免開始人擔心此語終將消逝。

世間事物種種，只要為人所用，自會綿延不息、流傳不止；反之，則如「學而不思則罔」之理，終將「不用則亡」：只要是日常生活，派不上用場之事物，自會被人淡忘，隨著歳月流逝而逐漸消亡。

時至今日，閩南話「有音無字」的困境，依然困擾著無數的使用人們。古人為求功名，勤讀詩書，故而漢字與閩南話之關聯，尚能自文讀音獲取，《廣韻》、《康煕字典》等古籍，更能為之佐證；然「白話音」則幾近空白，我手寫我口，目前仍是難以企及的美夢。

閩南話特有的四聲、八調，用於誦讀：古文、詩、詞，乃至經文，總能令人感受到一股醇厚雋永的風雅韻味。先祖所傳之漢字、雅音，此等彌足珍貴的文化遺産，盼能永世流傳，勿使斷絕！一介草民，無權、無財，所能者何？且從閩南話的漢字輸入法發展開頭吧！

### 輸入法引擎

專案所開發之輸入方案，以【中州韻輸入法引擎】（簡稱：RIME）作為輸入法作業平臺。RIME 因俱備跨越作業系統之特性；故而本專案産出之輸入法，可於 Windows, macOS, Linux 各類作業系統操作使用。

RIME 設定檔名稱為：`default.custom.yaml`，各作業系統之存放 `目錄路徑（資料夾）`，條列如下：

- 小狼毫(Windows)：`"%APPDATA%\Rime"`
- 中州韻(Linux)：`~/.config/ibus/rime/`
- 鼠鬚管(macOS)：`~/Library/Rime/`

## 安裝作業

安裝作業之主要流程，其執行要點摘要如下。如需操作細節詳述，請參考 [Wiki](https://github.com/AlanJui/rime-tlpa/wiki) 之安裝作業指引文件。

1. **下載及安裝 [RIME 中州韻輸入法引擎](http://rime.im)**。

   RIME 可於不同之作業系統運作，且各有其名稱，使用者該擇用何者，請參考如下之說明：
   - macOS: 鼠鬚管
   - Windows: 小狼毫
   - Linux: 中州韻 (ibus-rime, fcitx-rime, fcitx5-rime)

   【註】： 由於 Linux 作業系統發布版眾多（如：Ubuntu, LinuxMint, ArchLinux）...，
   且 Linux 的各個發布版又有不同的【輸入法框架】，如：iBus, Fcitx 和 Gcin...。
   目前 RIME 在 Linux 作業系統可運作的輸入法框架，有：ibus-rime, fcitx-rime
   及 fcit5-rime 共三種。所以，在安裝 RIME 之前，得先確認 Linux 作業
   系統安裝的【輸入法框架】為何者。


2. **下載 [輸入方案套件包](https://github.com/AlanJui/rime-tlpa/releases)。**

<img alt="image" src="https://github.com/user-attachments/assets/7d387177-9777-4ebf-8484-39673bdad2a5" />

- rime-tlpa-all-v0.1.zip: 專案産出所有的輸入方案（包函：反切/注音/拼音，所有的輸入方案）
- rime-tlpa-huan_ciat-v0.1.zip：【反切類輸入方案】壓縮包
- rime-tlpa-zu_im-v0.1.zip：【注音類輸入方案】壓縮包
- rime-tlpa-phing_im-v0.1.zip：【拼音類輸入方案】壓縮包

3. **下載及執行[安裝程式](https://github.com/AlanJui/rime-tlpa/releases)。**

下載輸入方案安裝程式執行檔： RIME_TLPA_Installer.exe

**【安裝程式注意要項】**：

- 安裝程式 `RIME_TLPA_Installer.exe` **不內含**輸入方案檔；請**先**下載上述
    `rime-tlpa-*.zip` 任一壓縮包（如僅需全部方案，下載 `rime-tlpa-all-*.zip` 即可）。
- 將下載的 `.zip` 壓縮檔放到**安裝程式（.exe）所在資料夾**，或保留在
    **【下載 (Downloads)】資料夾**即可——安裝程式啟動時會自動偵測這兩處
    （以及目前工作目錄）的 `rime-tlpa-*.zip`。
- 執行 `RIME_TLPA_Installer.exe` 後，於畫面上方的下拉選單**選擇要安裝的壓縮檔**，
    再點擊【開始安裝】；若壓縮檔放在其他位置，可點【瀏覽…】自行指定。
- 若未偵測到任何壓縮檔，安裝程式會中止並提示先行下載，請依提示完成下載後再重試。

### 羅馬拼音字母

【拼音輸入法】為求使用上之簡便，故以【台語拼音】之羅馬拼音字母作為輸入標準。（全名：
[臺灣語言音標方案；簡稱：TLPA）](https://zh.wikipedia.org/zh-tw/%E8%87%BA%E7%81%A3%E8%AA%9E%E8%A8%80%E9%9F%B3%E6%A8%99%E6%96%B9%E6%A1%88)。TLPA 是台灣語文學會於 1991 年期間，針對臺灣主要語言，所制定的
音標系統，適用於為：臺灣閩南語、臺灣客家語、臺灣原住民語標注讀音。之後教育部
更在 2003 年 2 月，將「閩南語」部份與「國際語音符號（國際音標）」、「教會羅馬字
」系統整合，隨後於 2006 年 10 月 14 日公布為「[臺灣閩南語羅馬字拼音方案（簡稱：
台羅拼音）](https://zi-hi.com/%E5%8F%B0%E7%BE%85%E6%8B%BC%E9%9F%B3)」。

## 方音符號與 Unidoce 編碼

### Unicode 編碼區段

「方音符號」有專屬的 Unicode 區段：

方音符號：U+31A0 ～ U+31BF

注音符號：U+3100 ～ U+312F

注音符號擴充：U+31A0 ～ U+31BF

台灣方音符號補充（部分符號會在 Unicode Extensions 或 CJK 擴充區裡）

### 瀏覽字型對方音符號的支援

某些字型不支援【方音符號】；或是會發顯示錯誤之問題。

- 無法顯示【台語注音調號】
  - 陰去調：˪ U+02EA MODIFIER LETTER YIN DEPARTING TONE MARK
  - 陽去調：˫ U+02EB MODIFIER LETTER YANG DEPARTING TONE MARK

- 無法顯示入聲韻尾符號/或是將 ㆻ 顯示成 ㆶ
  - ㆴ U+31B4 BOPOMOFO FINAL LETTER P
  - ㆵ U+31B5 BOPOMOFO FINAL LETTER T
  - ㆶ U+31B6 BOPOMOFO FINAL LETTER K
  - ㆷ U+31B7 BOPOMOFO FINAL LETTER H
  - ㆻ U+31BB BOPOMOFO FINAL LETTER G（收錄於2020年Unicode v.13，當下多數作業系統內建字體都尚未添加此字）
