## 建置 Python 作業環境

### 建置 Python 虛擬環境

在【專案根目錄】，建置 Python 虛擬環境 `.venv` 。

```bash
cd C:\Users\AlanJui\work\rime-tlpa
py -m venv .venv
```

---

### 需要安裝的 Python 套件

（從各 .py 檔的 import 掃描而來）
  | 套件名稱 | 用途 | 備註 |
  |----------|------|------|
  | `pandas` | 資料處理 | 需安裝 |
  | `openpyxl` | 讀寫 Excel `.xlsx` | 需安裝 |
  | `xlwings` | Excel 操作（需有 Excel） | 需安裝 |
  | `pypandoc` | 文件格式轉換 | 需安裝（且需安裝 Pandoc） |
  | `tkinter` | GUI 介面 | Python 內建，無需安裝 |
  | `pyinstaller` | 打包 `.exe` | 需安裝（見 `.spec` 檔） |

```powershell
 PS C:\Users\AlanJui\work\rime-tlpa> .\.venv\Scripts\Activate.ps1
(.venv) PS C:\Users\AlanJui\work\rime-tlpa> pip install -r .\requirements.txt
```

  注意：pypandoc 除了 pip 套件外，還需另外安裝 https://pandoc.org/installing.html
  本體才能正常運作。

  winget install --source winget --exact --id JohnMacFarlane.Pandoc

● 如果專案中有用到 pypandoc，就需要安裝 Pandoc 本體。

● 只有 src/make_install_sop.py
  一支程式用到。看檔名是製作安裝說明文件用的，屬於輔助工具，不是核心功能。

  結論：

  - 若不需要執行 make_install_sop.py，可以暫時跳過 winget install Pandoc
  - 若日後需要產生安裝說明文件，再補裝即可
