# 反切輸入法【方音符號】設計規格

`版本：V0.0.1`

## 摘要

### 特性說明

- 【輸入類型】：反切輸入法（採用：《彙集雅俗通十五音》）
- 【字典標準】：漢字拼音法之【台羅拼音】
- 【按鍵編碼】：以【台語音標】對映【十五音】之【聲】、【韻】、【調】
- 【候選清單】：採【雙欄】顯示
  1. 左欄為【十五音】
  2. 右欄為【方音符號】
- 【特性】包括：
  1. 支持15音調映射
  2. 多樣化拼音兼容性配置（方音與兩岸著重特徵）。

## 操作情境

漢字【忍】依【十五音】漢字標音法標示，其【音節】構成三元素，【韻母】+【聲調】+【聲母】為：【君】+【二】+【柳】；故其【十五音漢字標音】為【君二柳】。

- 韻母（韻）：君
- 聲調（調）：二
- 聲母（聲）：柳

【音節】三元素之輸入，以【**`方音符號`**】代：韻母、聲調、聲母。

### 1. 自鍵盤輸入`方音符號`

使用者在【輸入編輯列】，依【聲】、【韻】、【調】之順序，在鍵盤按鍵輸入【台語音標】之**方音符號**。

| # |鍵盤按鍵|方音符號|音節元素|十五音|
|---|:-----:|:----:|:----:|:----:|
| 1 |x      |ㄌ |聲 |柳|
| 2 |j      |ㄨ |韻 |  |
| 3 |p      |ㄣ |韻 |君|
| 4 |4      |ˋ  |韻 |四|

**`【註】`**：【**方音符號**】：`ㄨㄣ`，構成【**十五音之韻母**】：`君`。

### 2. 方音符號轉換成十五音之聲韻調

輸入方案自鍵盤接收到【按鍵】，交由【轉換作業】處理，先將【**鍵盤按鍵**】轉換成【**方音符號**】；然後再將【方音符號】轉換成`十五音`之【聲母】、【韻母】、【聲調】。

| # |按鍵|方音符號|聲/韻/調|
|:-:|:--:|------|:-----:|
| 1 |c  |ㄌ     | 柳 |
| 2 |jp |ㄨㄣ   | 君 |
| 3 |4  |ˋ     | 二 |

### 3. 在【輸入編輯列】顯示十五音之聲韻調

按鍵經**轉換處理作業**，依：【聲】、【韻】、【調】之順序，於【**輸入編輯列**】顯示`柳君二`。

![](./static/pict/輸入編輯列.png)

### 4. 在【候選清單】顯示符合十五音之漢字及漢字標音

【**候選清單**】依據【**輸入編輯列**】已取得之輸入，顯示符合【聲韻調】輸入之漢字及漢字標音。

**`【候選清單】顯示格式`**

採【雙欄】顯示：
  1. 左欄：【十五音】
  2. 右欄：【方音符號】

**候選清單範列**

1. 輪 〔柳君五〕 【 ㄌㄨㄣˊ】
2. 論 〔柳君七〕 【 ㄌㄨㄣ˫】
3. 潤 〔柳君七〕 【 ㄌㄨㄣ˫】
4. 忍 〔柳君二〕 【 ㄌㄨㄣˋ】
5. 怕 〔柳君二〕 【 ㄌㄨㄣˋ】

![](./static/pict/候選清單.png)

**`【註】`**：在【輸入編輯列】顯示之十五音，其【音節】格式為：**【聲】、【韻】、【調】**；然【候選清單】所顯示之十五音，則為傳統之十五音格式（**【韻】、【調】、【聲】**）。

### 5. 使用【選擇游標】選擇欲輸出之漢字

移動【**選擇游標**】，選擇欲輸出之【漢字】（或【音節】之【漢字標音】）。

【候選清單】中顯示之項目，可借由【PgUp】、【PgDn】、【↑】、【↓】鍵之操作來進行選取。本輸入方案仿 `Vim` 編輯器之 **hjkl** 按鍵，亦可用於操作【選擇游標】之移動。

- 翻到上一頁： [Ctrl] + [h]
- 翻到下一頁： [Ctrl] + [l]
- 移到上一個： [Ctrl] + [j]
- 移到下一個： [Ctrl] + [k]

- 移到中央處： [Ctrl] + [m]
- 移到底端處： [Ctrl] + [<]
- 移到頂端處： [Ctrl] + [>]

![](./static/pict/選擇游標.png)

### 6. 多音節連續輸入處理

漢字為：一字一音，本輸入方案提供【連續輸入】操作方式，供使用者連續輸入多個【音節】，一次完成多個【漢字】輸入。

如【辭彙】：「呑忍」之輸入操作...

|漢字|鍵盤按鍵|方音符號|十五音標音|
|:--:|:----:|:-------:|:----:|
|呑	|wjp'	|ㄊㄨㄣ¯    |他君一	|
|忍	|xjp4	|ㄌㄨㄣˋ    |柳君二	|

使用者欲完成「呑忍」之輸入，可用漢字逐一輸入方式完成；亦可使用【連續輸入】方式完成。

在中州韻(RIME)輸入法平台，其【連續輸入】操作步驟如下：

1. 輸入【辭彙】之第一字：在鍵盤按：【w】【j】【p】【'】鍵；

    - 【輸入編輯列】顯示： `【他君一】`；
    - 【候選字視窗】顯示： `呑〔君一他〕 【 ㄊㄨㄣˉ 】`。

2. 輸入【辭彙】第二字：不要按【空白】鍵，接繼在鍵盤按：【x】【j】【p】【4】鍵。

    - 【輸入編輯列】顯示： `【他君一 柳君二】`；
    - 【候選字視窗】顯示： `呑忍 〔君一他 君柳二〕 【 ㄊㄨㄣˉ ㄌㄨㄣˋ 】`

## 輸入方案設計規範

以下說明本輸入方案：如何於**中州韻(RIME)輸入法平台**，實作【輸入方案】之規範。

- 識別代碼：**`huan_ciat_tps`**
- 檔案名稱：**`huan_ciat_tps.schema.yaml`**

### 字典輸入與輸入方案編碼

**字典檔（.dict.yaml）**採用【漢字拼音法】之【台羅拼音】。

由於輸入方案，底層核心仍為：【台語音標】（TLPA+），故自【字典檔】讀入之【聲】、【韻】、【調】等【音節】資料，需進行【編碼轉換】，其作業需特別處理以下所列要項：

`台羅拼音轉台語音標作業要點`

1. 聲母轉換

- 雙唇音（清聲送氣塞音/清聲不送氣塞音/濁聲不送氣塞音/濁聲鼻音）：
  - ph → ㄆ [ph]
  - p → ㄅ [p]
  - b → ㆠ [b]
  - m → ㄇ [m] (鼻音)

- 軟顎音（清聲送氣塞音/清聲不送氣塞音/濁聲不送氣塞音/濁聲鼻音）：
  - kh → ㄎ [kh]
  - k → ㄍ [k]
  - g → ㆣ [k]
  - ng → ㄫ [ŋ] (鼻音)

- 齒齦音（清聲送氣塞音/清聲不送氣塞音/濁聲邊音/濁聲鼻音）：
  - th → ㄊ [th]
  - t → ㄉ [t]
  - l → ㄌ [l]（邊音）
  - n → ㄋ [n] (鼻音)

- 齒齦音（清聲送氣塞擦音/清聲不送氣塞擦音/濁聲不送氣塞擦音/清聲擦音）：
  - c → ㄘ [tsh]
  - z → ㄗ [ts]
  - j → ㄌ [d͡z]
  - s → ㄙ [s] (擦音)

- 齦顎音（清聲不送氣塞擦音/清聲送氣塞擦音/濁聲不送氣塞擦音/清聲擦音）：
  - ci → ㄑㄧ [t͡ɕʰ]
  - zi → ㄐㄧ [t͡ɕ]
  - ji → ㆢㄧ [d͡ʑ]
  - si → ㄒㄧ [ɕ] (擦音)

  【齒齦音/舌齒音】聲母，後接【韻母】：**i** 時（或簡稱：**介音顎化**）：當聲母為 `z, c, s, j` ，且其後接 `i` 系列韻母時，需轉換為顎化符號（`ㄐ, ㄑ, ㄒ, ㆢ`）。
  【齒齦音 + i】轉換成【齦顎音 + i 】規則：
  - j+i: ㆡ[j] + ㄧ[i] = ㆢ[J] + ㄧ[i]
  - z+i: ㄗ[z] + ㄧ[i] = ㄐ[Z] + ㄧ[i]
  - c+i: ㄘ[c] + ㄧ[i] = ㄑ[C] + ㄧ[i]
  - s+i: ㄙ[s] + ㄧ[i] = ㄒ[S] + ㄧ[i]

- 聲母音
  - h → ㄏ [h] (清聲擦音)
  - (零聲母) → （無對映之方音符號，故不標示） [ʔ]

2. 韻母轉換

- **韻母變更**：
  - a → ㄚ
  - i → ㄧ
  - u → ㄨ
  - e → ㆤ
  - oo → ㆦ [ɔ]
  - o → ㄜ [ə]
  - ai → ㄞ（不要轉成：ㄚㄧ）
  - au → ㄠ（不要轉成：ㄚㄨ）
  - am → ㆰ（不要轉成：ㄚㆬ）
  - ang → ㄤ（不要轉成：ㄚㆭ）
  - an → ㄢ（不要轉成：ㄚㄣ）
  - om → ㆱ（不要轉成：ㆦㆬ）
  - ong → ㆲ（不要轉成：ㆦㆭ）
  - ing → ㄧㄥ [iəŋ]（不要轉成：`ㄧㆭ`）

    **【註】：其它注意要點**
    - 韻尾 `-n`，轉成 `ㄣ` ，但只用在結合韻母：ㄧㄣ、ㄨㄣ及ㆨㄣ。

- **韻化輔音變更**：
  韻化輔音指**亦可當作【韻母】使用的【聲母】（亦稱：輔音）**

  - aunn → ㆯ
  - ann → ㆩ
  - inn → ㆪ
  - unn → ㆫ

### 【輸入編輯列】之漢字標音格式

使用者在【輸入輯輯列】完成應有之輸入，再搭配【選擇游標】之移動擇定欲輸出之漢字後，最後按下【空白鍵】，則本輸入方案，將輸出【漢字】。

顧及使用者有時需要的【輸出】不是【漢字】，而是【漢字標音】（如：十五音、方音符號、台語音標...等），故輸入方案提供`【輸出漢字標音選項】`。供使用者設定【輸出】使用之【漢字標音】標準。而輸出之按鍵，使用【Enter】鍵。

#### `【需求詳述】`

以漢字：【忍】舉例，其【漢字標音】可為以下任何一種方式呈現：

|漢字標音法|英文簡稱|拼音/注音/十五音標音|
|---------|------|--------------|
|台語音標   |TLPA   |lun2|
|十五音     |SNI    |柳君二|
|方音符號   |TPS    |ㄌㄨㄣˋ|
|台羅拼音   |TL     |lún|
|閩拼方案   |BP     |lún|

`操作步驟`：

1. 使用者輸入漢字之【台語音標】：lun2；

    - 【鍵盤按鍵】：【t】【h】【u】【n】【;】
    - 【輸入編輯列】顯示： `【柳君二】`
    - 【候選清單】挑選： `忍 〔lun2〕 【ㄌㄨㄣˋ】`

2. 若按【空白】鍵，輸出漢字：【忍】；若按【Enter】鍵，輸出【十五音】：`【柳君二】`。

3. 將`【輸出漢字標音選項】`自預設之【十五音】，切換成【台語音標】；

4. 重覆 2. 步驟之操作，則可於按下【Enter】鍵後，得到輸出【台語音標】：`lun2`。

5. 將`【輸出漢字標音選項】`，切換成【方音符號】；

6. 重覆 2. 步驟之操作，則可於按下【Enter】鍵後，得到輸出【方音符號】：`ㄌㄨㄣˋ`。

### 【候選清單】之漢字標音格式

【候選清單】中顯示之【漢字標音】，可由**候選清單之漢字標音格式選項**，控制【候選清單】之【顯示內容】。

![](./static/pict/候選清單之漢字標音格式.png)

- switches/options: hau_suan_piau_im_*

- 選項：
   | states | options |
   |:-------|:--------|
   |方音符號（預設）| hau_suan_piau_im_tps |
   |台語音標 | hau_suan_piau_im_tlpa |

- 使用插件：
    engine/filters: lua_filter@reformat_comment_filter

### 函式：format_comment(comment_string, mode)

變更雙欄候選清單中，左欄之漢字標音、右欄之漢字標音。

- 因【輸入方案】支援【連續輸入】，取漢字之音節，需考量音節數不一定為單一

- 左欄之漢字標音：
    - 漢字標音法採：`十五音`，各音節以符號`〔...〕`包裹標示；
    - 由【輸入方案（YAML）】傳來之原始音節格式為：【聲+韻+調】，需變更成：【韻+調+聲】格式；
    【例】：忍【lun2】--> [l + un + 2] --> 【柳+君+二】--> 【柳君二】==> 【居二柳】

- 右欄之漢字標音：
    - 漢字標音法採：`方音符號`，各音節以符號`【...】`包裹標示；
    - 右欄漢字標音：非固定，依【**候選清單之漢字標音格式選項**】之設定變更。
        - 預設為：方音符號 (hau_suan_piau_im_tps)，可由【輸入方案】傳入，無需進行轉換；
        - 若切換成：台語音標 (hau_suan_piau_im_tlpa)，則需進行漢字標音轉換，改成【台語音標】。
