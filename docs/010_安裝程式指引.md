# 安裝程式指引 v0.2

## 簡介

專案透過之 RIME 輸入法引擎，開發之【閩南話輸入方案】，需要有個【安裝】程式，簡化使用者的操作。雖說安裝作業，可人工手動完成。但是，要使用者透過【檔案總管】，自行操作檔案的【複製】、【貼上】，將輸入方案使用的檔案，放入 RIME 的設定目錄路徑，如：Windows 11 為 $HOME\AppData\Roaming\Rime\ ，總有些不便；且容易出錯。

使用【PowerShell Script】製作【安裝程式】，對開發者而言是輕鬆且不費事的作法；但是，顧及使用者要在【終端機】執行 PowerShell Script 構成的【安裝程式】，有技術門檻高度，令使用者犯難。因為極可能有人連【終端機】是什麼東西都不知道，更遑論要他能執行 PowerShell 執行的操作。

基於上述考量，本專案的【安裝程式】為便利使用者，也需簡化程式編撰過程。故採用 Python 模組：PyInstaller 製作【安裝程式】，將 Python Script: rime_installer.py 加一外層，封裝成 .exe 執行檔。如此一來，使用者只需在【檔案總管】，雙擊（Double Click）封裝成 .exe 檔的安裝程式，便能完成輸入方案之安裝。

## 安裝程式架構及使用組件

### 安裝程式架構

採【Python + tkinter GUI + PyInstaller】架構，將【安裝程式底層】包裝成 .exe 執行檔。

- 安裝程式：此為【安裝程式】之核心邏輯，檔名為：rime_installer.py；
- 進度狀態：為令使用者掌握安裝程式之執行進度，使用 tkinter 供使用者觀察安裝進度；
- 打包封裝：使用 PyInstaller 負責將 Python Script 之安裝程式，打包封裝成 .exe 執行檔。

```powershell
pyinstaller .\RIME_TLPA_Installer.spec
```

產出之執行檔置於： .\dist\RIME_TLPA_Installer.exe。

### 安裝程式底層（核心邏輯）

使用 Python Script 編撰【安裝程式】，檔名：rime_installer.py 。


## 安裝程式

檔名：.\tools\rime_installer.py

此為【安裝程式】之核心邏輯。執行時期需搭配【封裝描述檔】： release_include.txt ，以便掌握打包
時，需納入封裝的檔案有那些。

### 設計規格

1. RimeTLPAInstaller — 新增 log_func 參數（預設 print，CLI 模式完全相容），所有 print() 改為 self._log()

2. InstallerApp（新增） — tkinter 圖形介面，包含：
    - 深色主題標題列
    - 彩色記錄視窗（綠色成功 / 黃色警告 / 紅色錯誤）
    - 不定進度條（安裝期間自動轉動）
    - 「開始安裝」按鈕（安裝中自動停用，完成後恢復）
    - 安裝完成後跳出 messagebox 提示
    - 背景執行緒安裝，GUI 不會凍結

3. RIME_TLPA_Installer.spec — console=True → console=False（視窗模式，不顯示黑色終端機）

4. 輸出 — RIME_TLPA_Installer.exe，約 11.9 MB，單一 .exe 無依賴，使用者 Double Click 即可安裝。

## 打包封裝作業

使用 PyInstaller ，執行安裝程式及相關檔案之打包封裝作業。

搭配使用之輸入檔案：
- RIME_TLPA_Installer.spec

執行輸出結果：
- RIME_TLPA_Installer.exe

### 工作目錄

PyInstaller 在執行時期，使用之工作目錄如下：

|目錄   |PyInstaller 用途|
|-------|----------------|
|build	|暫存工作目錄，存放編譯中間產物（.toc、.pkg 等），每次重建都會覆蓋|
|dist	|最終輸出目錄，存放打包好的 .exe，也屬於編譯產物，不應納管|

上述之 PyInstaller 工作目錄，無需使用 git 進行版本管控，故需將之加入 .gitignore。

### 打包作業

將 Python Script 寫成之【安裝程式】，打包封裝成 .exe 執行檔，使用如下指令達成：

```powershell
(.venv) PS C:\Users\AlanJui\work\rime-tlpa> pyinstaller RIME_TLPA_Installer.spec
```

或

```powershell
(.venv) PS C:\Users\AlanJui\work\rime-tlpa> py -m PyInstaller .\RIME_TLPA_Installer.spec
```
