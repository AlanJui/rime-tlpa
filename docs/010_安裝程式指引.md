# 安裝程式指引

## 簡介

專案透過之 RIME 輸入法引擎，開發之【閩南話輸入方案】，需要有個【安裝】程式，簡化使用者的操作。雖說安裝作業，可人工手動完成。但是，要使用者透過【檔案總管】，自行操作檔案的【複製】、【貼上】，將輸入方案使用的檔案，放入 RIME 的設定目錄路徑，如：Windows 11 為 $HOME\AppData\Roaming\Rime\ ，總有些不便；且容易出錯。

使用【PowerShell Script】製作【安裝程式】，對開發者而言是輕鬆且不費事的作法；但是，顧及使用者要在【終端機】執行 PowerShell Script 構成的【安裝程式】，有技術門檻高度，令使用者犯難。因為極可能有人連【終端機】是什麼東西都不知道，更遑論要他能執行 PowerShell 執行的操作。

基於上述考量，本專案【安裝程式】製作的想法為：【安裝程式】底曾仍是以 PowerShell Script 完成（名稱為： deploy_rime.ps1），然後將 PowerScript 加一外層，包裹成 .exe 執行檔。如此一來，使用者只需在【檔案總管】，雙擊（Double Click）被包裹成 .exe 檔的安裝程式，便能完成輸入方案之安裝。

## 安裝程式組件

### 安裝程式底層

使用 Windows PowerShell Script 達成，檔名：deploy_rime.ps1

### 安裝程式殼層

採【Python + tkinter GUI + PyInstaller】架構，將【安裝程式底層】包裝成 .exe 執行檔。

- 核心還輯：rime_installer.py；
- 進度狀態：使用 tkinter 可供使用者觀察安裝進度；
- 程式打包：使用 PyInstaller 負責打包成一個 .exe 執行檔。


```powershell
pyinstaller RIME_TLPA_Installer.spec
```

產出在 dist/RIME_TLPA_Installer.exe，使用者直接 Double Click 即可。


## 核心邏輯

rime_installer.py — 核心邏輯不動，加入兩個新元素：

1. RimeTLPAInstaller — 新增 log_func 參數（預設 print，CLI 模式完全相容），所有 print() 改為 self._log()
2. InstallerApp（新增） — tkinter 圖形介面，包含：
    - 深色主題標題列
    - 彩色記錄視窗（綠色成功 / 黃色警告 / 紅色錯誤）
    - 不定進度條（安裝期間自動轉動）
    - 「開始安裝」按鈕（安裝中自動停用，完成後恢復）
    - 安裝完成後跳出 messagebox 提示
    - 背景執行緒安裝，GUI 不會凍結

RIME_TLPA_Installer.spec — console=True → console=False（視窗模式，不顯示黑色終端機）

輸出 — RIME_TLPA_Installer.exe，約 11.9 MB，單一 .exe 無依賴，使用者 Double Click 即可安裝。

## rime_installer.py

### 使用時機
rime_installer.py 使用時機有兩個：

#### 時機一：開發 / 測試階段（開發者）**

```powershell
# 在專案虛擬環境中直接執行，測試邏輯是否正確python tools\rime_installer.py
此時會以 CLI 模式（print 輸出）或 GUI 模式（tkinter 視窗）執行，不需打包，方便快速驗證。
```

#### 時機二：打包成 .exe（開發者）

```powershell
# 在專案虛擬環境中直接執行，測試邏輯是否正確python tools\rime_installer.py
pyinstaller RIME_TLPA_Installer.spec
PyInstaller 讀取 rime_installer.py 作為入口點，將它連同 Python 執行環境一起打包進 RIME_TLPA_Installer.exe。打包完成後，rime_installer.py 的原始碼已「嵌入」.exe 內，使用者永遠不會碰到這個 .py 檔。
```

### 整體流程圖

開發者                              使用者  │                                   │  ├─ 編輯 tools/rime_installer.py     │  ├─ python tools/rime_installer.py   │  ← 開發測試  ├─ pyinstaller RIME_TLPA_Installer.spec  │        │  │        └─→ dist/RIME_TLPA_Installer.exe  │                                   │  │          發佈 .exe 給使用者 ──────►├─ Double Click RIME_TLPA_Installer.exe  │                                   └─ 安裝完成

**結論**：rime_installer.py 只有開發者才會用到，是打包的原料；使用者拿到的只有 RIME_TLPA_Installer.exe。
