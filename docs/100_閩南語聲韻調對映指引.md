# 閩南語聲韻調對映指引 V0.1
---

本文件說明【閩南語】標注漢字讀音使用之【聲母】、【韻母】與【音調】，如何映對及互換。

涉及之漢字標音標準有：

- 國際音標（IPA）
- 十五音（SNI）：此處之【十五音】指《彙集雅俗通十五音》
- 方音符號（TFS）
- 台語音標（TLPA+）：以下簡稱：TLPA
- 台羅拼音（TL）
- 白話字（POJ）
- 閩拼方案（BP）
- 台語注音二式（BPM2）：以下用【注音二式】表【台語注音二式】

## 設計要點

**algebra 核心流程（9 步驟）：**

| 步驟 | 處理                              | 範例                       |
| ---- | --------------------------------- | -------------------------- |
| 1    | 聲母 TLPA→BP（有序）              | `tsh→c`, `b→bb` 先於 `p→b` |
| 2    | 韻母 TLPA→BP                      | `ir→eu`, `au→ao`, `ann→na` |
| 3    | 聲調轉換 `xlit\|234567\|352468\|` | TLPA 3 → BP 5              |
| 4    | 變調規則（BP 調號）               | `1→6→5→3→2→6`              |
| 5    | 入聲縮寫                          | `[ptkh]7→[ptkh]9`          |
| 6    | y/w 零聲母相容                    | `ia→ya`, `u→wu`            |
| 7    | 聲母縮寫                          | 只打 `bb` 找帽類字         |
| 8    | 無調輸入                          | `bbna` 找含「媽」字        |
| 9    | xlit BP 調號→按鍵                 | `1→;`, `6→-`, `5→_`        |

**完整輸入流程範例（媽 mah1）：**

| 位置               | 字串        |
| ------------------ | ----------- |
| 字典原碼           | `mah1`      |
| 聲母轉換           | `bbnah1`    |
| 聲調轉換           | 不變（1→1） |
| prism 儲存         | `bbnah1`    |
| 使用者鍵入 `bbna;` | `bbnah1` ✓  |
| preedit xlit 逆轉  | `bbnah1`    |
| 調符移位           | `bbna1h`    |
| `xform/a1/ā/`      | `bbnāh` ✓   |
